# 教育培训管理系统

#### 介绍
教育培训机构的客户管理crm + 在线视频学习管理+ 题库和考试作业+ 购买课程管网

#### 软件架构

本系统采用前后端分离的开发和运行模式。前端使用vue框架并基于vue-admin-element 模板进行开发。后端使用golang + redis + mysql 进行搭建。 对中小型培训机构的使用没有压力。经实测4vCPU+8G内存的服务器
